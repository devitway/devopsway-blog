{{- /* Series Navigation with Progress — показывает все посты серии и текущую позицию */ -}} {{- with .Params.series -}} {{- $currentSeries := . -}} {{- $currentPage := $ -}} {{- /* Собираем все посты серии, сортируем по weight → date */ -}} {{- $seriesPosts := where site.RegularPages ".Params.series" $currentSeries -}} {{- $seriesPosts = sort $seriesPosts ".Params.seriesWeight" "asc" -}} {{- if gt (len $seriesPosts) 1 -}} {{- /* Вычисляем позицию текущего поста */ -}} {{- $total := len $seriesPosts -}} {{- $currentIdx := 0 -}} {{- range $i, $p := $seriesPosts -}} {{- if eq $p.Permalink $currentPage.Permalink -}} {{- $currentIdx = add $i 1 -}} {{- end -}} {{- end -}} {{- /* Название серии из params или fallback */ -}} {{- $seriesTitle := $currentPage.Params.seriesTitle | default (printf "Серия: %s" $currentSeries) -}}

    {{- range $i, $p := $seriesPosts -}} {{- $isCurrent := eq $p.Permalink $currentPage.Permalink -}} {{- $isPast := lt (add $i 1) $currentIdx -}}
